### 4. Create SSH Key Pair (Secure Authentication)

SSH avoids typing username/password every time.

Generate key:

```bash
ssh-keygen -t ed25519 -C "youremail@example.com"
```

ðŸ‘‰ Creates **private key (local)** and **public key (to upload)**

Show public key:

```bash
cat ~/.ssh/id_ed25519.pub
```

---

### 5. Add SSH Public Key to GitHub / GitLab

* Go to **Settings â†’ SSH Keys**
* Paste the public key

ðŸ‘‰ Enables secure connection between local machine and remote repo.

Test connection:


### 6. Configure Git Username & Email

This identifies **who made the commit**.

```bash
git config --global user.name "Dev Kumar Sen"
git config --global user.email "dev@email.com"
```

Verify:

```bash
git config --list
```

---

### 7. Clone Remote Repository

Copy the remote repo to your local system.

