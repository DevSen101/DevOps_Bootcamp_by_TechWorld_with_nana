### Deleting Branches (Local & Remote)

**Purpose:** Clean up branches after work is merged.

#### Delete Local Branch

```bash

```

ğŸ‘‰ Deletes branch **only if merged**.

Force delete (not merged):

```bash

```

#### Delete Remote Branch

```bash
git push origin --delete feature-login
```

ğŸ‘‰ Removes branch from GitHub/GitLab.

**Verify**


**Best Practice:** Delete feature branches after merge to keep repo clean.

## Resolving Merge Conflicts

**When it happens:** Same lines edited in different branches.

#### Conflict Example

```txt
<<<<<<< HEAD
console.log("Main branch code");
=======
console.log("Feature branch code");
>>>>>>> feature-login
```

#### Steps to Resolve

1. Open the conflicted file
2. Choose correct code (or combine)
3. Remove conflict markers
4. Mark resolved:

```bash
git add filename
```

#### Continue Rebase / Merge

* During rebase:

```bash
git rebase --continue
```

* During merge:

```bash
git commit
```

**Abort if needed**

```bash
git rebase --abort
git merge --abort
```
